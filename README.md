# 한글다운 (HangeulDown)

**한글 문서 작성을 위한 현대적인 마크다운 에디터**

> 🚀 **순수 HTML/CSS/JavaScript로 제작** - 별도 빌드 과정 없이 바로 호스팅!

## ✨ 특징

### 📝 강력한 편집 기능
- **CodeMirror** 기반 에디터 (VS Code 수준)
- **실시간 미리보기** 지원
- **GitHub Flavored Markdown** 완벽 지원
- **로컬 스토리지** 자동 저장

### 🎨 사용자 경험
- **다크/라이트 테마** 전환
- **키보드 단축키** 지원
  - `Ctrl+B`: 굵게
  - `Ctrl+I`: 기울임
  - `Ctrl+K`: 링크
  - `Ctrl+S`: 저장
  - `Ctrl+O`: 열기
  - `Ctrl+P`: 인쇄

### 📊 고급 기능
- **수식 지원**: KaTeX로 LaTeX 수식 렌더링
- **테이블 생성**: 원클릭 테이블 삽입
- **코드 하이라이팅**: 다양한 언어 지원
- **파일 작업**: 저장, 불러오기, HTML 내보내기
- **인쇄 최적화**: 전문적인 인쇄 출력

## 🛠️ 기술 스택
- **HTML5**: 시맨틱 마크업
- **CSS3**: 모던 스타일링 (CSS Variables, Flexbox)
- **Vanilla JavaScript**: 프레임워크 없는 순수 JS
- **CodeMirror**: 고급 코드 에디터
- **Marked.js**: 마크다운 파싱
- **KaTeX**: 수식 렌더링
